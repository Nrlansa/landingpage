Apa itu SiAbsensi?

Siabsensi merupakan sebuah sistem berbasis web yang membantu melihat kehadiran seseorang dalam jangka waktu tertentu. Sistem ini dapat melakukan beberapa hal secara fleksibel salah satunya adalah dapat mengekspor data dalam bentuk excel.

Efektif

Sistem ini menghadirkan cara absensi yang lebih cepat, cukup dengan bermodalkan gadget yang setiap hari anda gunakan, anda sudah bisa mengakses sistem ini dan menggunakannya.



Sistem Berbasis Web

Sistem berbasis website ini dirancang agar lebih mudah diakses, pada zaman serba modern hal ini menjadi penting bagi anda yang sudah terbiasa dengan informasi digital.



Qr Code

SIABSENSI adalah sistem yang menyediakan Bar Code untuk mengisi kehadiran anda, hanya dengan menampilkan layar gadget mengisi absen menjadi lebih keren.



Mempermudah Perekapan Data

Sistem ini cocok untuk anda yang setiap harinya berurusan dengan berkas absensi maupun data lainnya, waktu anda akan lebih efisien dengan perekapan yang lebih cepat dibuat.

Fitur Absensi

SiAbsensi menyediakan fitur fitur yang dapat membantu dan memudahkan sesorang sebagai berikut.

Lupa Password

Merupakan menu bantu bagi user untuk menghubungi seorang admin ketika user tidak mengingat Password yang telah dibuat.



Import File

Merupakan menu yang dapat membantu seseorang dalam mengambil file excel dari luar aplikasi tanpa harus menuliskan data satu persatu.



Absensi

Merupakan menu yang dapat mempermudah dalam melihat kehadiran sesorang dalam jangka waktu tertentu.
